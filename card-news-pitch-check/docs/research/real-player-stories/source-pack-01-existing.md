# 기존 출처 팩 정규화 기록

이 기록은 이미 수집된 두 개의 URL 단위 출처 팩을 카탈로그로 정규화한 내용입니다.

- [실제 선수 스토리 Scrapling 출처 팩](../../real-player-story-scrapling-source-pack.md)
- [Ian Wright 교사 재회 출처 팩](../../ian-wright-teacher-source-pack.md)

정규화한 항목:

- 레거시 스토리 뱅크의 `sourceRefs` 22개 ID와 원본 URL
- URL의 정확한 호스트명에 따른 발행처와 신뢰 등급
- 수집된 제목, HTTP 상태, 미디어 후보, 출처 팩 파일 경로
- 발행일 미확인 값은 `null` 처리
- 기존 편집 출처의 사용 상태를 `reference-only`로 통일하고 권리 확인 문구 추가

원문 기사나 영상 설명의 긴 문단은 이 문서에 복사하지 않습니다. 본문은 증거 확인에만 사용하고, 공개 카드에는 확인된 사실을 짧게 재서술합니다. 미디어 후보도 권리가 확인되기 전에는 직접 사용하지 않습니다.
