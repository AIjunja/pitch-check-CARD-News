# 실제 선수 스토리 출처 인덱스

이 폴더는 PitchCheck 실제 선수 스토리 작업에 쓰는 출처의 **인덱스**입니다. 정규화된 22개 출처는 [카탈로그](../../../samples/pitchcheck/real-player-source-catalog-300.json)에서 `sourceId`, URL, 발행처, 신뢰 등급, 수집 상태와 미디어 후보로 확인합니다.

Scrapling은 URL 단위의 출처 팩 보조 도구로만 사용합니다. 넓은 범위의 수집, 대량 스크래핑, 접근 제한 우회에는 사용하지 않습니다. 이 저장소의 팩은 이미 수집된 URL을 정리한 것이며, 새 수집을 뜻하지 않습니다.

출처의 본문은 사실 확인을 위한 증거입니다. 카드 문구나 공개용 카피로 긴 문단을 그대로 복사하지 말고, 필요한 사실을 짧게 재서술하고 원문 URL을 남깁니다.

`mediaCandidates`는 시각 검증을 위한 후보일 뿐입니다. 직접 사용 전에는 매체별 권리, 저작자 표시, 사용 문맥과 라이선스를 확인해야 하며, 확인 전 상태는 모두 `reference-only`입니다.

정리 기록은 [source-pack-01-existing.md](source-pack-01-existing.md)를 봅니다.
